# Dataset

The dataset used for this project is hosted on Kaggle:

**LaLiga Teams Logo Dataset**  
https://www.kaggle.com/datasets/tomasawad/laliga-teams-logo

## Notes

- The dataset contains LaLiga team logo images used for image classification.
- The app predicts 40 football club logo classes.
- Keep the class order in `class_names.json` exactly the same as the training order when retraining or replacing models.
